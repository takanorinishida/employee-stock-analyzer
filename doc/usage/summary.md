# 保有状況確認・売却シミュレーション

---

## summary — 保有状況サマリー

```
stock summary PLAN_ID
```

指定した持株会の現在の保有状況を表示します。

### 引数

| 引数 | 説明 |
|---|---|
| `PLAN_ID` | 確認する持株会の ID |

### 出力例

```
保有株数          : 15.0000
平均取得単価(奨励込): 1097.50 円
平均取得単価(拠出のみ): 1000.00 円
累計拠出金額      : 20000 円
累計実現損益(奨励込): 1012 円
累計実現損益(拠出のみ): 1500 円
```

### 出力項目の説明

| 項目 | 説明 |
|---|---|
| 保有株数 | 現在保有している株数（小数点以下 4 桁） |
| 平均取得単価(奨励込) | 奨励金・配当を原価に含めた加重平均単価 |
| 平均取得単価(拠出のみ) | 自己拠出金のみで計算した加重平均単価 |
| 累計拠出金額 | これまでに拠出した合計金額 |
| 累計実現損益(奨励込) | 売却済み分の損益合計（奨励金・配当を原価に含む） |
| 累計実現損益(拠出のみ) | 売却済み分の損益合計（自己拠出のみ） |

2 種の平均取得単価については [基本概念](concepts.md) を参照してください。

---

## simulate — 売却シミュレーション

```
stock simulate PLAN_ID --price PRICE [--shares SHARES]
```

指定した株価で売却した場合の損益・税額・手取り額を試算します。

### 引数

| 引数 | 説明 |
|---|---|
| `PLAN_ID` | シミュレーション対象の持株会の ID |

### オプション

| オプション | 必須 | 説明 |
|---|---|---|
| `--price DECIMAL` | 必須 | 想定売却単価（円/株） |
| `--shares DECIMAL` | 任意 | 売却株数（省略時は全保有株数） |

### 実行例

```bash
# 全株を 1,500 円で売却した場合
stock simulate toyota --price 1500

# 10 株だけ 1,500 円で売却した場合
stock simulate toyota --price 1500 --shares 10
```

### 出力例

```
シミュレーション株数  : 15.0000
売却益(奨励込)       : 6037 円
売却益(拠出のみ)      : 7500 円
推定税額(奨励込)      : 1226 円
推定税額(拠出のみ)    : 1523 円
税引後手取り(奨励込)  : 21311 円
税引後手取り(拠出のみ): 20977 円
```

### 出力項目の説明

| 項目 | 説明 |
|---|---|
| シミュレーション株数 | 試算対象の売却株数 |
| 売却益(奨励込/拠出のみ) | 売却総額 − 平均取得単価 × 株数（1 円未満切り捨て） |
| 推定税額 | 売却益 × 税率（1 円未満切り捨て） |
| 税引後手取り | 売却総額 − 推定税額 |

税率は `config.json` の `capital_gains_tax_rate` が使用されます（デフォルト: 20.315%）。
詳細は [インストール・初期設定](install.md) を参照してください。
